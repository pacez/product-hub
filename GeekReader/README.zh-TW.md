# GeekReader

> 一款純粹電子書閱讀器

[下載 App Store 版本](https://apps.apple.com/app/geekreader) · [問題反饋](https://github.com/pacez/product-hub/issues)
-------------------------------

## 簡介

**GeekReader** 是一款專為 Mac 設計的電子書閱讀與管理工具，致力於為你提供簡潔、沉浸、高效的閱讀體驗。支持 EPUB、PDF、TXT 等多種格式，同時具備連接 GitHub 遠程書源的能力，讓你輕鬆打造個人專屬書庫。

---

## 核心功能

### 📚 多格式本地閱讀

支持 EPUB、PDF、TXT 等主流電子書格式，無論是小說、技術文檔還是學術論文，都能輕鬆打開閱讀。

### 📥 便捷導入

- 拖拽導入：直接將書籍文件拖入應用窗口即可
- 文件選擇器：批量選擇本地書籍一鍵導入

### 🌐 在線書源

支持添加 GitHub 公開倉庫作為遠程書源，自動解析目錄結構，在線瀏覽和下載書籍。

### 🔍 智能書庫管理

- 網格 / 列表雙視圖自由切換
- 按全部、最近閱讀、收藏、在讀、已讀、未讀快速篩選
- 實時搜索書名與作者
- 批量管理書籍

### 🎨 個性化排版

- **字體**：New York、SF Pro、Source Han Serif
- **字號**：12pt ~ 32pt 無級調節
- **行高**：1.0 ~ 3.0 自由調整
- **主題**：日間、夜間、羊皮紙、跟隨系統

### 🌍 多語言支持

簡體中文、繁體中文、English，一鍵切換。

## 常見問題

### 如何導入書籍？

點擊工具欄的導入按鈕（或按 ⌘N），選擇本地 EPUB / PDF / TXT 文件即可。也支持直接從 Finder 拖拽文件到應用窗口。

### 如何添加遠程書源？

在設置中選擇「在線書源」標籤頁，輸入 GitHub 公開倉庫地址和書源名稱即可。應用會自動解析倉庫中的書籍文件。

### 閱讀進度會同步嗎？

閱讀進度自動保存在本地，目前不支持跨設備同步。你可以通過導出/導入配置文件手動遷移數據。

### 支持哪些電子書格式？

當前支持 EPUB、PDF、TXT。後續版本計劃支持更多格式。

### 如何切換語言？

在偏好設置 → 通用中，可選擇跟隨系統、簡體中文、繁體中文或 English。

## 隱私政策

GeekReader 尊重並保護所有用戶的個人隱私。

- **不收集個人數據**：我們不會收集、存儲或分享任何可識別個人身份的信息。
- **本地存儲**：所有書籍文件、閱讀進度、批註筆記均存儲在您的設備本地。
- **無追蹤**：應用中不包含任何第三方追蹤或分析工具。
- **遠程書源**：遠程書源內容由用戶自行配置的第三方 GitHub 倉庫提供，開發者無法訪問、控制或審查這些內容。
